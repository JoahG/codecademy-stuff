SCT Viewer

This JavaScript bookmarklet is based on the original Submission Correctness Viewer bookmarklet by minrice (Michael Tackes). My version should work with Codecademy’s new coding UI.

It uses Twitter Bootstrap’s Modal component (the new UI already loads the CSS styles for that; the javascript part is loaded dynamically on first use).

The bookmarklet only works on Codecademy exercise pages (like this one). Drag the link below to your bookmarks (keeping it in the Bookmarks bar gives you one-click access), then use it in any Codecademy exercise.

Show SCT

There may be bugs and I haven’t tested it on any browser other than the latest Chrome for Mac. So if you find any errors, do let me know.
